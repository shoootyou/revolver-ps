Bulk Upload Videos using CSV file

By using this action, you can bulk upload videos through csv file. You can provide either file path or folder path in csv file to upload videos. Office 365 Video does not support video of size more than 2 GB

Note: Your csv file should contains the column headers ChannelName, FilePath, Title and Description

Sample CSV file: 
ChannelName,FilePath,Title,Description
"Tech Support","C:\TechVideos\testvid.wmv","Test Video","This is test video"
"Sales Support","C:\SalesVideos\testvid1.wmv","Test Video 1","This is test video 1"
"Sales Support","C:\SalesVideos\testvid2.wmv","Test Video 2","This is test video 2"
"Demo Videos","C:\Demo\AllVideos",,
  • ChannelName - Name of the channel to upload videos.
  • FilePath - Video file path or folder path that contains the videos.
  • Title - Title of the video.(Leave empty if not required to set)
  • Description - Description of the video.(Leave empty if not required to set)

    Manage Channel Permissions using CSV file

By using this action, you can manage/update channel permissions through csv file. You can add or remove permisions from a particular channel, to add you need to set the specified permission column as true, to remove set it as false, otherwise leave it as empty.

Note: Your csv file should contains the column headers ChannelName, UserEmail, Owners, Editors,Viewers

Sample CSV file: 
ChannelName,UserEmail,Owners,Editors,Viewers
"Tech Support",Admin@MODXXXX.onmicrosoft.com,true,true,true
"Demo Videos",Admin@MODXXXX.onmicrosoft.com,true,true,true
"Sales Support",Admin@MODXXXX.onmicrosoft.com,true,,
"Sales Support",AlexD@MODXXXX.onmicrosoft.com,,true,
"Sales Support","Everyone except external users",,,true
"Demo Videos",SaraD@MODXXXX.onmicrosoft.com,false,false,false
  • ChannelName - Name of the channel to update permission.
  • UserEmail - User email, or group name to add or remove permission.
  • Owners - Set ture to add user as Owner, or set false to remove user from Owners group.
  • Editors - Set ture to add user as Editor, or set false to remove user from Editors group.
  • Viewers - Set ture to add user as Viewer, or set false to remove user from Viewers group.

    Bulk Channel Creation using CSV file

By using this action, you can create bulk video channels and set channel permissions through csv file.

Note: Your csv file should contains the column headers ChannelName, Owners, Editors, Viewers

Sample CSV file: 
ChannelName,Owners,Editors,Viewers
"Tech Support","Admin@MODXXXX.onmicrosoft.com;AlexD@MODXXXX.onmicrosoft.com","Admin@MODXXXX.onmicrosoft.com","Everyone except external users"
"Sales Support","Admin@MODXXXX.onmicrosoft.com;AlexD@MODXXXX.onmicrosoft.com","SaraD@MODXXXX.onmicrosoft.com","Everyone except external users"
"Demo Videos","Admin@MODXXXX.onmicrosoft.com","Admin@MODXXXX.onmicrosoft.com;AlexD@MODXXXX.onmicrosoft.com","Everyone"
  • ChannelName - Name of the channel to update permission.
  • Owners - User email, or group name to add Owners.
  • Editors - User email, or group name to add Editors.
  • Viewers - User email, or group name to add Viewers.

Note: You can give multiple names as semi-colon (;) separeted values for Owners, Editors and Viewers
