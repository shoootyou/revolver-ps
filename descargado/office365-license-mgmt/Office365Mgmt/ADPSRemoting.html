How to Configure Remote PowerShell ?

Before adding AD Domain, follow the below steps to enable Remote PowerShell on both local machine (where GingerEx tool installed) and remote server machine (AD Domain Controller).

CASE 1:

If your computers (local machine and AD Domain Controller) are in the same domain network, then run the following powershell command to enable PS Remoting in the remote system.

1. Enable-PSRemoting -Force
2. Restart-Service WinRm

CASE 2:

If your computers (local machine and AD Domain Controller) are in different domain or workgroup, then you need to configure the Trusted hosts settings in both the systems.

Run the following command in PowerShell (both local machine and AD Domain Controller).

1. Enable-PSRemoting -Force
2. Set-Item wsman:\localhost\client\trustedhosts -Value <name_of_the_other_system>
3. Restart-Service WinRm
